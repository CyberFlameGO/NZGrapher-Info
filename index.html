This Site contains links to the NZQA and TKI websites to help find the info for NCEA easily, as well as supporting material for NZGrapher. To see the old materials that used to be on here go to https://old.mathsnz.com and https://students.mathsnz.com. The resources there are no longer being maintained, but are provided as is, where is, if anyone still does want to use them.

My wife Olivia and I have also created a site that covers the Base Maths Concepts Linked to the Learning Progression Framework. This is a free resource for anyone to use to help them with their planning and teaching.
